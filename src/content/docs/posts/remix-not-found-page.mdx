---
title: Remix Not Found Page
---

루트 에러 바운더리를 사용할때 존재하지 않는 페이지 방문 시 루트 에러바운더리가 이를 처리합니다.

근데 이는 로더를 활용할 수 없기 때문에 입력한 URL마다 서로 다른 메시지를 출력할 수 없음.

이 문제는 스플랫 라우트로 해결. 즉, `$.tsx` 경로 파일을 사용함.

이 파일에서 로더를 활용해 응답을 던지고, 에러 바운더리가 이를 받고, 페이지 라우트가 에러 바운더리를 렌더링하는 방식 사용.

스플랫 라우트는 모든 경로를 의미하지만, 우선순위가 가장 낮으므로 존재하지 않는 경로들에 매핑 됨.
