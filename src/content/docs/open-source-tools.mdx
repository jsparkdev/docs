웹 개발을 하면서 유용하게 사용중인 오픈 소스 도구들을 소개합니다.

## 릴리스 및 버전 관리

- [pkg.pr.new](https://github.com/stackblitz-labs/pkg.pr.new): 패키지의 프리뷰 릴리스를 빠르게 확인할 수 있는 도구입니다.
- [changesets](https://github.com/changesets/changesets): 모노레포 패키지의 버전을 쉽게 관리할 수 있는 도구입니다.

## 빌드 도구

- [tsdown](https://github.com/rolldown/tsdown): 간단한 구성의 라이브러리 번들러입니다.

## 코드 품질

- [publint](https://github.com/publint/publint): `package.json` 파일의 품질을 검사할 수 있는 도구입니다.

## Git 훅 매니저

- [lefthook](https://github.com/evilmartians/lefthook): 빠르고 간편하게 사용할 수 있는 Git 훅 매니저입니다.

## 패키지 매니저

- [pnpm](https://github.com/pnpm/pnpm): 모노레포 또는 패키지를 효율적으로 관리할 수 있는 패키지 매니저입니다.
- [bun](https://github.com/oven-sh/bun): 성능이 뛰어난 패키지 매니저입니다.

## 유틸리티

- [giget](https://github.com/unjs/giget): 원격 리포지토리를 다운로드할 수 있는 CLI와 API를 제공하는 도구입니다.
- [@bluwy/giget-core](https://github.com/bluwy/giget-core): CLI를 제거하여 더 간소화된 [giget](https://github.com/unjs/giget)입니다.

## Github Actions

- [checkout](https://github.com/actions/checkout)
- [pnpm](https://github.com/pnpm/action-setup)
- [changesets](https://github.com/changesets/action)
- [Node.js](https://github.com/actions/setup-node)
